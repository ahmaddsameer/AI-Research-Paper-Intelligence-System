# Project Summary

This project was completed as part of my AI & Machine Learning internship.

The objective of this project is to simplify research paper discovery using Natural Language Processing (NLP). The system performs semantic search using Sentence Transformers and FAISS, extracts important keywords, identifies named entities using a BERT-based NER model, and generates concise summaries using BART.

The project demonstrates the practical application of semantic search, transformer models, and NLP techniques for academic literature exploration.
